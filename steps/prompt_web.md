## 2. Prompt for Web Development
```
// 1
After click cart to see all products in cart to open path /cart
That /cart page use selected frame in pencil with react and tailwindcss to show all products in cart


// 2
After click proceed to checkout button to open path /checkout
That /checkout page use selected frame in pencil with react and tailwindcss to show checkout form and order summary.


// 3
After filling out the checkout form and clicking the "Complete Order" button, the user will be redirected to a confirmation page at path /confirmation.
The /confirmation page will display a summary of the order, use selected frame in pencil with react and tailwindcss to show the order details and a thank you message to the user for their purchase.

```
